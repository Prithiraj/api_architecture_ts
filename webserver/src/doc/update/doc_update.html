API Documentation for PUT for general purpose

Description

  1. The PUT APIs in documentation update your existing record based on groupd or specific parameters such as primary keys. To operate this API primary key is always a required field.

  2. For successful updation, any one of the key must be provided.

  3. This documentation is for the general purpose use. This means, user will be able operate only on the informaiton that the user him/herself created it.
